# Product Kanban Demo

Trang demo Vue 3 (script setup) cho bố cục Kanban quản lý sản phẩm.

## Yêu cầu
- Node.js 18+ và npm

## Cài đặt & chạy
1. Cài dependencies:
   ```bash
   npm install
   ```
2. Chạy dev server (Vite):
   ```bash
   npm run dev
   ```
   Mặc định chạy ở http://localhost:5173
3. Build production:
   ```bash
   npm run build
   ```
4. Xem bản build:
   ```bash
   npm run preview
   ```

## Cấu trúc
- `src/` chứa App Vue và các component Kanban
- `index.html` là entrypoint Vite
- `vite.config.js` cấu hình plugin Vue
